A stack is a linear data structure with 3 basic operations:

  • push: add an item to the stack
  • pop: remove an item from the stack
  • top: return the top item from the stack

These operations are performed based on either a LIFO (Last In, First Out) or FIFO (First In, First Out) order.

There are two ways to implement a stack: with an array or a linked list!
