Linked list is another linear data structure. A linked list, like an array, is also a collection of items.

Instead of being stored in contigious memory locations, elements of a linked list can be stored anywhere in memory.

Advantages Disadvantages
Useful when the number of objects is unknown Does not allow random access
Easy to add and delete items Uses more memory than an array
